# 无服务器调度系统

## 介绍
这是一个基于无服务器架构的调度系统。它支持用户创建和分配任务，进行实时进度跟踪，并通过电子邮件和消息通知团队成员。

## 功能
- 用户注册和身份验证
- 创建、编辑和删除任务
- 任务分配给团队成员
- 实时进度跟踪
- 自动通知功能（电子邮件、内部消息）

## 技术栈
- AWS Lambda
- API Gateway
- DynamoDB
- SNS 或 SQS 用于消息传递

## 快速开始
1. 克隆这个仓库： `git clone https://github.com/your-username/serverless-scheduling-system`
2. 安装依赖项
3. 部署到云服务
